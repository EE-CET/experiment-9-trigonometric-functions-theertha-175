# Experiment 9: Trigonometric Functions

## Problem Statement

Write a program to find the sine and cosine of a given angle in degrees.
You must create two separate methods:
1. `public static double sine(double angle)`
2. `public static double cosine(double angle)`

**Note:** Java's `Math` library (e.g., `Math.sin`, `Math.cos`) expects angles in **radians**. You will need to convert the degrees to radians first using `Math.toRadians()`.

## Input Format

The first line contains a number representing the angle in degrees.

## Output Format

Print the sine and cosine values in the format shown below.

### Example 1

**Input:**

```text
0
```

**Output:**

```text
Sine: 0.0
Cosine: 1.0
```

### Explanation

* Sine of 0 degrees is 0.0.
* Cosine of 0 degrees is 1.0.
